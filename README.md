# Infinity.Build

Build Unity3d project to ios, android, osx, win64 platforms.

# Install

```
git submodule add git@bitbucket.org:networm/infinity.build.git Infinity.Build
```

# Usage

```
cd Infinity.Build
ruby build.rb
```

# Help

```
cd Infinity.Build
ruby build.rb --help
```
